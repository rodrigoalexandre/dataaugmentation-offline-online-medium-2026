## 📝 **DESCRIÇÃO**

Este repositório contém o projeto criado para o hands-on apresentado no artigo **"Abordagens de Data Augmentation para Deep Learning: conceitos básicos e implementação"** publicado no blog Medium em maio de 2026.

O artigo apresenta uma introdução às abordagens off-line e on-line de data augmentation e uma demonstração do uso prático destas abordagens para aumentar um dataset para o treinamento de um modelo de Deep Learning.
<br><br>
## 📂 **ESTRUTURA DO PROJETO**

|-- notebook    Jupyter notebook com o pipeline de treinamento do modelo e arquivos de configuração \
|-- dataset   Dataset utilizado no treinamento \
